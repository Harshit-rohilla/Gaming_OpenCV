# Gesture Gaming
The project contains four games controlled by hand gestures.In this user have to select any one game and can play the game by different hand gestures.  
  
Four games are-  
1. Flappy Bird  
2. Rock aper Scissors  
3. Snake Game  
4. Car Racing
